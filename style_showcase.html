Position

First of all, look at a top right corner of a window. See my message? That's a fixed position option which puts a fixed position on a screen for an element. Therefore, it stays even if you scroll the screen.

That's how they are supposed to be placed by default, and that position is called static. Now let's put the second element off of normal layout using relative option position:

Also we can use absolute position which fixes an element relative to it's closest positioned parent with excluding it from a normal layout for other elements.
